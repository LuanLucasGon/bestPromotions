# bestPromotions

> Projeto desenvolvido exclusivamente para fins de estudo, aplicação de conceitos e novos aprendizados. Não destinado a uso em produção.

## Tecnologias

| Tecnologia | Versão  |
|------------|---------|
| Go         | 1.26    |
| Angular    | 21      |

## Sobre o Projeto

Este repositório é um ambiente de experimentação pessoal para explorar e consolidar conhecimentos em desenvolvimento full-stack com **Go** no backend e **Angular** no frontend.

O foco está no aprendizado prático de:

- Construção de APIs REST com Go
- Componentização e reatividade com Angular
- Comunicação entre frontend e backend
- Boas práticas de estrutura de projeto

## Pré-requisitos

- [Go 1.26+](https://go.dev/dl/)
- [Node.js 22+](https://nodejs.org/)
- [Angular CLI 21](https://angular.dev/tools/cli)

## Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/bestPromotions.git
cd bestPromotions
```

### Backend (Go)

```bash
cd backend
go mod tidy
go run main.go
```

### Frontend (Angular)

```bash
cd frontend
npm install
ng serve
```

## Aviso

Este projeto é de uso estritamente educacional. O código pode conter experimentos, testes e implementações inacabadas como parte natural do processo de aprendizado.
